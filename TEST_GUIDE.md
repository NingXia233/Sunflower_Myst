# 向日葵之谜 - 测试指南 (Test Guide)

## 功能测试清单

### 基本功能测试
- [ ] 页面正常加载，显示游戏界面
- [ ] 可以看到一个向日葵在墙壁中央
- [ ] 可以看到5个红色圆圈标记（角落）
- [ ] 右上角显示太阳，并且太阳在旋转
- [ ] 左侧UI显示正确的统计信息（向日葵数量：1，覆盖角落：0/5）

### 交互功能测试
- [ ] **点击向日葵**：
  - 点击向日葵后应该看到蓝色泪滴从向日葵掉落
  - 泪滴落地后应该在新位置生长出新的向日葵
  - 向日葵数量应该增加
  - 被点击过的向日葵不能再次流泪

- [ ] **旋转墙壁**：
  - 按住鼠标拖拽应该能改变墙壁的视角
  - 墙壁应该有3D透视效果
  - 左右拖拽应该改变墙壁的水平旋转
  - 上下拖拽应该改变墙壁的垂直旋转

- [ ] **覆盖角落**：
  - 当向日葵靠近角落时，角落应该变成绿色
  - 覆盖角落的计数应该增加
  - 当所有角落都被覆盖时，应该显示胜利消息

### 胜利条件测试
- [ ] 当所有5个角落都被向日葵覆盖时：
  - 显示胜利消息弹窗
  - 消息中包含"恭喜过关！"
  - 显示"重新开始"按钮

- [ ] 点击"重新开始"按钮后：
  - 所有向日葵被清除，只剩一个初始向日葵
  - 角落位置重新随机生成
  - 统计信息重置
  - 可以再次正常游戏

### 视觉效果测试
- [ ] 太阳有发光效果和旋转的光芒
- [ ] 向日葵有花瓣和中心细节
- [ ] 向日葵会轻微旋转动画
- [ ] 泪滴有半透明效果和拖尾
- [ ] 墙壁有逼真的砖块纹理
- [ ] 背景有渐变的天空效果

### 浏览器兼容性测试
- [ ] Chrome/Edge
- [ ] Firefox
- [ ] Safari
- [ ] 移动设备浏览器（触摸屏支持）

### 性能测试
- [ ] 页面加载速度快（无需下载外部库）
- [ ] 动画流畅，无卡顿
- [ ] 多个向日葵（10+）时性能良好
- [ ] 内存使用稳定

## 游戏玩法测试示例

### 测试场景1：快速通关
1. 打开游戏
2. 观察5个角落的位置
3. 策略性地点击向日葵，让新向日葵生长在角落附近
4. 重复直到所有角落被覆盖
5. 验证胜利消息显示

### 测试场景2：墙壁旋转
1. 打开游戏
2. 用鼠标拖拽墙壁，向左右旋转
3. 验证墙壁宽度有透视变化
4. 向上下旋转，验证视角变化
5. 在旋转的状态下点击向日葵，验证游戏逻辑仍然正常

### 测试场景3：多次重新开始
1. 玩完一局游戏
2. 点击重新开始
3. 验证角落位置改变
4. 验证游戏状态完全重置
5. 再玩一局验证功能正常

## 已知限制
- 向日葵只能流一次泪
- 角落位置在每局游戏中随机生成
- 拖拽和点击操作互斥（拖拽时不会触发点击）
