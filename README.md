# Sunflower Myst

A puzzle game where the sun makes sunflowers cry, and their tears bring new life to the ruins.

## How to Play

1. **Run the Game:**
   * **Option 1 (Python):** Run the `start_game.py` script.

     ```bash
     python start_game.py
     ```

   * **Option 2 (VS Code):** Install the "Live Server" extension, right-click `game/index.html` and select "Open with Live Server".

2. **Controls:**
   * **Left Click:** Click on a sunflower to make it cry.
   * **Right Click + Drag:** Rotate the 3D view to find hidden corners.
   * **Goal:** Spread the sunflowers across the dilapidated walls.

## Concept

Sunflowers grow on a 3D wall structure. When they cry, their tears fall. If a tear hits a suitable spot on the wall below, a new sunflower grows.
